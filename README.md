# 开眼
一个使用 Flutter 开发的 开眼 第三方客户端应用，提供了 开眼 App 的主要功能和内容展示。

## 开始使用

### 环境要求

- Flutter 3.29.0 或更高版本
- Dart 3.7.0 或更高版本
- DevTools 2.42.2 或更高版本

### 安装步骤

1. 克隆项目
```bash
git clone https://github.com/JWInCoding/flutter_eyepetizer.git
```

2. 进入项目目录
```bash
cd flutter_eyepetizer
```

3. 获取依赖
```bash
flutter pub get
```

4. 生成多语言文件
```bash
flutter gen-l10n
```

5. 运行项目
```bash
flutter run
```
